# Kin Keeper

## 📌 Project Overview
Kin Keeper is a relationship management web application built to help users maintain meaningful personal connections. It allows users to track friendships, log interactions, monitor contact goals, and get reminders when it's time to reconnect.

## 🚀 Short Description
Kin Keeper helps users nurture friendships intentionally by organizing contacts, tracking communication history, and providing simple tools for regular check-ins through calls, texts, or video interactions.

## 🛠 Technologies Used
- Next.js (App Router)
- React
- Tailwind CSS
- Context API
- Recharts
- React Toastify
- JSON Data

## ✨ Key Features

### 1. Friend Relationship Tracking
- Add and manage friend profiles
- Track contact frequency goals
- View friend status (On-Track, Almost Due, Overdue)

### 2. Quick Check-In & Timeline
- Log Call, Text, and Video interactions
- Automatically generate timeline entries
- Filter timeline by interaction type

### 3. Friendship Analytics Dashboard
- Visual pie chart analytics by interaction type
- Contact statistics and relationship goals
- Responsive dashboard UI with status cards

## 📱 Responsive Design
- Mobile friendly
- Tablet optimized
- Desktop dashboard layout

## 🔔 Additional Features
- Toast notifications for check-ins
- Dynamic friend detail pages
- Tag-based friend organization
- Clean modern UI
